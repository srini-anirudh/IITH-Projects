# DevRev_2k23
## Repository Notes

**Project type:** RAG research prototype

**Summary:** Retrieval augmented generation experiments for tool/document workflows, including evaluation metrics and LLM notebooks.

**How to use:** Use the existing README, dataset folder, and metric scripts to reproduce retrieval/evaluation runs.

**Layout:** Source code, notebooks, datasets, reports, media, and generated assets are kept in their original project-relative folders so existing paths continue to work. Nested Git metadata and local build/cache outputs have been removed for clean monorepo versioning.

## Current Layout

- `src/` - LLM and fine-tuning code.
- `data/` - prompts, datasets, tool definitions, and splits.
- `experiments/` - zero-shot and few-shot experiment notebooks.
- `evaluation/` - metric and evaluation scripts.
- `docs/report/` - reports, paper assets, and presentation material.
