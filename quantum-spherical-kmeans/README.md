# ML_Repo
## Repository Notes

**Project type:** Quantum ML research code

**Summary:** Quantum spherical k-means experiments implemented with Python scripts and notebooks.

**How to use:** Run main.py or the notebooks after installing the Python dependencies used by the scripts.

**Layout:** Source code, notebooks, datasets, reports, media, and generated assets are kept in their original project-relative folders so existing paths continue to work. Nested Git metadata and local build/cache outputs have been removed for clean monorepo versioning.

## Current Layout

- `src/quantum_spherical_kmeans/` - reusable clustering implementation and experiment entrypoint.
- `notebooks/` - exploratory qMeans notebooks.
- `figures/` - generated plots.
- `docs/` - report PDF.
- `scripts/` - shell runner scripts.
